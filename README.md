# VaultLookup

A premium offline Excel and CSV search engine with instant lookup, multi-file indexing, and secure local-only access.

## Features

* Instant customer search
* Multi-file Excel support
* Works fully offline
* No cloud uploads
* Password protected
* CSV export
* Fast local indexing

## Supported Files

* `.xlsx`
* `.xls`
* `.csv`

## Live Demo

Open here:

https://damehedi.github.io/VaultLookup/

## Usage

1. Open the website
2. Upload Excel or CSV files
3. Search instantly across all records

## Privacy

All data stays inside your browser.
Nothing is uploaded to any server.
